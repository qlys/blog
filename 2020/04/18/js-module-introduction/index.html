发表于 2020-04-18 更新于 2020-04-19 分类于 JavaScript 阅读次数： 本文字数： 3.4k 阅读时长 ≈ 6 分钟

ES Module模块引入方式

ES Module模块引入在浏览器端模块需要提前打包处理编译(浏览器不支持ES Module模块引入),但是,也是最流行的模块引入方式.

CommonJS模块引入

CommonJS在服务器端的加载是同步的, 在浏览器端模块需要提前打包处理编译(浏览器不支持CommonJS规范),浏览器加载时可能微慢

AMD

专门用于浏览器端,模块的加载方式是异步.

AMD模块加载是异步的,jQuery支持AMD,但是jQuery引用时这步define(['module','jQuery'],function(module,$){})必须改成小写的jquery才行,就是这样define(['module','jquery'],function(module,$){})才不会报错;还有angular,js需要单独配置,正确配置如下:
